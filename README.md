sitemap
=======

crawl
=====

### install
```shell
go install github.com/dreamsxin/go-sitemap/cmd/crawl@latest
```

### usage
```shell
crawl -u http://example.com -o sitemap.xml
crawl -u http://example.com -o sitemap.xml -c 1
```
